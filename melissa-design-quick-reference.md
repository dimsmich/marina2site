## SPACING SCALE

```
SECTION SPACING (Vertical):
Mobile:   60px padding top/bottom
Desktop:  80-100px padding top/bottom

ELEMENT SPACING:
Between major elements:  30-40px
Between paragraphs:      16-24px
Between sections:        0 (use section padding)

GRID GAPS:
2-column: 60px
3-column: 40px
4-column: 30px

CONTAINER PADDING (Horizontal):
Mobile:   20-30px
Desktop:  50-100px
```

## IMAGE TREATMENTS

```
ROTATION ANGLES:
Product images: -6deg, -2deg, 5deg, 10deg (for variety)
Avatars: 0deg (no rotation)

FILTERS:
Blog images: grayscale(50%) → grayscale(0%) on hover
Logo cloud: grayscale(100%) opacity(60%) → full color on hover

ASPECT RATIOS:
Landscape: 1.5:1 (3:2)
Portrait:  0.67:1 (2:3)
Video:     1.78:1 (16:9)
Avatar:    1:1 (square/circle)

SHADOWS:
Subtle:    0 2px 10px rgba(0,0,0,0.05)
Medium:    0 10px 30px rgba(0,0,0,0.15)
Large:     0 20px 60px rgba(0,0,0,0.2)
```

## RESPONSIVE BREAKPOINTS

```
MOBILE FIRST STRATEGY:

< 768px (Mobile):
- Single column layouts
- 320px design width
- Font sizes: -20-30%
- Spacing: -30-50%
- Stack all grids
- Hamburger menu

≥ 768px (Desktop):
- Multi-column layouts
- 1200px max container
- Full font sizes
- Full spacing
- Horizontal menu
- Grid layouts active
```

## COMMON MISTAKES TO AVOID

```
❌ Using too many font families (stick to 3 max)
❌ Inconsistent spacing between sections
❌ Poor color contrast (check WCAG compliance)
❌ Missing hover states on interactive elements
❌ Forgetting mobile optimization
❌ Overcomplicated animations (keep it smooth)
❌ Too many competing CTAs
❌ Skipping image optimization
❌ No clear visual hierarchy
❌ Missing alt text on images

✓ Maintain consistent spacing scale
✓ Test on actual devices
✓ Use semantic HTML
✓ Optimize images before upload
✓ Clear primary CTA per section
✓ Subtle, purposeful animations
✓ Strong contrast for readability
✓ Clear focus states for accessibility
```
